Sincronização entre Computadores

Conecte a uma conta gratuita do Firebase para usar os mesmos dados em qualquer computador (você abre o app em outra máquina, com a mesma configuração, e os dados já estão lá). Como pegar essa configuração?

Sincronização desligada — dados salvos só neste navegador.
  1. Acesse console.firebase.google.com e crie um projeto gratuito (sem cartão de crédito).
  2. No menu lateral, vá em Realtime Database → Criar banco de dados → inicie em modo de teste.
  3. Vá em Configurações do Projeto (ícone de engrenagem) → role até "Seus apps" → clique em </> (Web) e registre um app.
  4. Copie o objeto firebaseConfig mostrado e cole no campo acima.
  5. Repita esses mesmos passos (ou apenas cole a mesma configuração) em cada computador que for usar o sistema.

Zona de Risco

Apaga permanentemente todos os produtos, containers, movimentações e lançamentos de consumo — útil para limpar dados de teste antes de começar a usar com dados reais. Exporte um backup antes, caso queira manter algum registro.
